BonkMenu is a comprehensive mod menu with direct game system manipulation. Features 200+ items/weapons/tomes/passives with instant spawning, full stat control, enemy modifiers, debuff systems, encounter spawning, loot rarity forcing, unlock automation, and player customization. Built on custom UniverseLib with deep IL2CPP integration for seamless runtime control.

====================================
DETAILED DESCRIPTION
====================================

BonkMenu is a feature-rich, professionally designed mod menu that provides complete control over game mechanics through direct runtime manipulation. Built on a custom-tailored UniverseLib framework with deep IL2CPP integration, this mod offers unprecedented access to game systems.

---
PLAYER TAB - Complete Character Control
---
• Infinite Resources: Unlimited refreshes, banishes, skips, and gold
• God Mode: Invincibility toggle for complete damage immunity
• Movement Control: Adjustable movement speed multipliers
• No Cooldowns: Instant ability recharge
• Stats System: 30+ individual stat sliders including:
  - Health, Armor, Regeneration, Evasion, Shield
  - Damage, Attack Speed, Critical Chance/Damage, Lifesteal
  - Movement Speed, Pickup Range, Projectile attributes
  - XP Gain, Gold multipliers, Luck modifiers
  - Advanced stats: Thorns, Knockback, Reroll Count, Curse effects
• Status Effect Buffs: Apply beneficial effects instantly
• Utility Functions: Heal to full, level up, prestige controls
